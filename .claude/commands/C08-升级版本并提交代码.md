# C08-升级版本并提交代码

## 项目规则

- **S03-版本发布规则**：`.cursor/rules/S03-版本发布规则.mdc`
- 构建系统：`.cursor/rules/S02-构建系统规则.mdc`
- 项目上下文：`.cursor/rules/S00-Claudian-工作区上下文.mdc`

## 项目配置（本仓库）

- 包管理：**`npm`**
- 版本同步：`npm version` 会触发 `scripts/sync-version.mjs` 同步 `manifest.json`
- 构建：`npm run build` → **`dist/claudian/`**

---

## 目标

按 **S03** 完成版本号、CHANGELOG、构建验证与本地提交；**默认不 push**。

## 执行步骤

1. `git status`（必要时 `git diff`），理解变更。
2. `npm version patch`（或 `minor` / `major`），按语义选择。
3. 更新根目录 **`CHANGELOG.md`（中文）**。
4. `npm run build`，失败则修复后重试。
5. `git add .`，`git commit -m "v{版本号} {简要说明}"`。
6. 仅当用户明确要求时执行 `git push`。
