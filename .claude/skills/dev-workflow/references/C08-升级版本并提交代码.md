# C08 升级版本并提交代码

## 项目规则

执行本指令时请参考：`.cursor/rules/S03-版本发布规则.mdc`、`.cursor/rules/S02-构建系统规则.mdc`、`.cursor/rules/S00-Claudian-工作区上下文.mdc`。

## 项目配置（Claudian）

- 包管理：**`npm`**
- `npm version` 会通过 `scripts/sync-version.mjs` 同步 `manifest.json`
- 构建：**`npm run build`** → `dist/claudian/`

## 执行步骤

1) `git status`（必要时 `git diff`）。

2) `npm version patch`（或 minor/major）。

3) 更新根目录 **`CHANGELOG.md`（中文）**。

4) `npm run build`，失败则修复。

5) `git add .`，`git commit -m "v版本号 简要说明"`。

6) 仅当用户明确要求时 `git push`。
