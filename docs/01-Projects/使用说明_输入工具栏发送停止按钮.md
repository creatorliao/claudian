# 使用说明：输入工具栏「发送 / 停止」按钮

本文说明 Claudian 聊天**输入区上方工具栏**中的圆形按钮：在**未生成回复**时用于**发送**，在**模型正在流式输出**时用于**停止**。与按 **Enter** 发送、按 **Esc** 中断流式输出走的是同一套逻辑，只是多了一条**看得见、可点击**的操作路径。

---

## 适用场景

- 习惯用鼠标或触控操作，不想记快捷键。
- 需要**明确知道「现在点的是发送还是停止」**（按钮图标会随是否在输出而切换）。
- 平板、触屏设备上，点按钮比键盘组合更顺手。

---

## 按钮在哪里

1. 打开 **Claudian** 聊天（侧边栏或独立标签/窗口均可）。
2. 看向**输入框上方**那一排工具栏（模型、权限、外部上下文等控件所在行）。
3. 其中有一个**圆角方形、品牌色背景**的小按钮：
   - **未在输出时**：图标为**向上箭头**，表示**发送当前输入**。
   - **正在流式输出时**：图标变为**小方块**，表示**停止当前生成**。

具体颜色随主题中的 **`--claudian-brand`** 与 Claudian 样式一致。

---

## 操作步骤

### 发送消息

1. 在输入框里写好内容（可照常附带 `@` 文件、图片、外部路径等）。
2. 确认按钮显示为**向上箭头**（说明当前不在流式输出中）。
3. **单击**该按钮。  
   效果与在输入框内按 **Enter** 发送（在符合当前输入规则的前提下）一致。

### 停止生成

1. 在模型**正在逐字输出**回复时，按钮会自动变为**方块**图标。
2. **单击**该按钮，即可**中断**当前这次流式输出。  
   效果与在聊天焦点下按 **Esc** 中断流式输出一致（见下文「与快捷键的关系」）。

---

## 与快捷键的关系

| 操作         | 按钮     | 常见快捷键（若当前焦点在聊天输入且规则允许）                             |
| ------------ | -------- | ------------------------------------------------------------------------ |
| 发送         | 向上箭头 | **Enter**（多行输入时可能是 **Ctrl/Cmd + Enter**，以你当前版本行为为准） |
| 停止流式输出 | 方块     | **Esc**（Claudian 视图内注册的快捷键，用于 `cancelStreaming`）           |

按钮不会取代快捷键，而是**并列选项**：任选其一即可。

---

## 常见情况说明

| 情况                               | 说明                                                                                                           |
| ---------------------------------- | -------------------------------------------------------------------------------------------------------------- |
| **按钮一直是箭头，点不了「停止」** | 说明当前没有在流式输出；只有生成过程中才会变成方块。                                                           |
| **点了发送没有反应**               | 检查输入是否为空、是否被某模式（如计划模式、特殊命令）限制发送；与不用按钮时的发送规则相同。                   |
| **点了停止，已打出的一部分文字**   | 一般会保留在消息里，仅中断后续生成；具体以当前 Provider（Claude / Codex 等）行为为准。                         |
| **多标签会话**                     | 每个标签有各自的输入区与工具栏按钮；停止只作用于**当前正在输出的那一轮**，切换标签时注意当前活跃的是哪个会话。 |

---

## 与 Provider 的关系

- 发送、停止都通过 Claudian 现有的 **`InputController`**（发送消息 / 取消流式）完成，**不单独实现一套流协议**。
- 因此 **Claude、Codex** 等已支持流式与取消的提供商，在界面上都会看到相同的按钮行为；若某场景下提供商本身不支持取消，停止效果以该运行时为准。

---

## 若界面里没有看到按钮

- 确认已更新到包含该功能的插件构建，并已**重新加载** Obsidian 或重新启用插件。
- 尝试**加宽** Claudian 面板：极窄时工具栏可能换行或被挤到可视区域外，可拖动分割线或全屏侧栏再查看。
