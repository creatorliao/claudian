# 溯源：Claude Code CLI 路径智能解析

## 功能摘要

- `findClaudeCLIPath`：合并 **设置 PATH**、`getExtraBinaryPaths()`、**进程 PATH** 后再探测；Unix 侧增加与 Windows 对齐的 **`cli.js` 推导**；补充 Homebrew、**pnpm 全局** 等候选；Windows 仍 **优先** 固定 `claude.exe` 位再合并 PATH。
- **`getExtraBinaryPaths`** 从 `env` 模块导出供共用。
