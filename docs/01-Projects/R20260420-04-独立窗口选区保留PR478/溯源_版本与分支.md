# 溯源：独立窗口 / 单独标签下保留编辑器选区（PR #478）

## 上游来源

- **Pull Request**：[YishenTu/claudian#478](https://github.com/YishenTu/claudian/pull/478) — `fix(selection): preserve selection when Claudian is in detached window or own tab`
- **关联 issue**：[#399](https://github.com/YishenTu/claudian/issues/399)（PR 说明中引用）

## 功能摘要

- 在 Claudian 置于 **独立窗口** 或 **单独标签页** 时，用户从笔记中划词后点击 Claudian，**对话侧的选区上下文**不再被轮询逻辑误清。
- 实现要点：`SelectionController` 增加 **`ownViewType`**；`isFocusWithinChatSidebar()` 在原有 DOM 焦点判断之外，若 **当前活动 leaf 的视图类型** 与 Claudian 视图一致，则视为仍处「本插件侧焦点语境」，从而避免在 Markdown 视图失焦、`activeElement` 为 `<body>` 时误判。
